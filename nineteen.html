第十九课——修改属性以及发送ActionBar ​

属性（Attributes）是原版系统。下面我们通过插件来修改玩家的属性。
玩家移动时随机修改玩家的近战攻击伤害这个属性在100至200之间，玩家攻击生物时将伤害值显示在经验值上面的消息栏：

java
@EventHandler
    public void onPlayerMove(PlayerMoveEvent event) {
        Player player = event.getPlayer();
        // 随机生成100-200之间的攻击力属性值
        Random rand = new Random();
        int damage = rand.nextInt(101) + 100;
        // 设置玩家近战攻击伤害属性值
        player.getAttribute(Attribute.GENERIC_ATTACK_DAMAGE).setBaseValue(damage);
    }

    @EventHandler
    public void onEntityDamageByEntity(EntityDamageByEntityEvent event) {
        Entity damager = event.getDamager();
        if (damager instanceof Player) {
            Player player = (Player) damager;
            // 获取玩家当前近战攻击伤害属性值
            double damage = player.getAttribute(Attribute.GENERIC_ATTACK_DAMAGE).getValue();
            // 在经验值上显示攻击伤害
            player.sendActionBar(ChatColor.RED + String.format("伤害值：%.2f", damage));
        }
    }

​​nextInt()​是Java中Random类的一个方法，用于生成一个随机数。nextInt(int bound)方法会返回一个随机的整数，范围为 ​0​到 ​bound-1​之间。例如，​nextInt(10)​会生成一个在 ​0​到 ​9​之间的随机整数。
​​rand.nextInt(101)​这个方法会返回一个0到100之间的随机整数。“​​+ 100”​这个操作会将生成的随机整数加上100，从而得到一个100到200之间的随机整数。

getAttribute(Attribute.GENERIC_ATTACK_DAMAGE)​会返回一个Attribute对象。其中，Attribute.GENERIC_ATTACK_DAMAGE​表示近战攻击伤害属性。
​​setBaseValue(damage)​会将近战攻击伤害属性的基础值设置为指定的数值 ​damage​。
在Minecraft中，所有属性都有一个基础值和一些额外的修饰值，最终的属性值等于基础值加上所有修饰值的总和。因此，通过 ​setBaseValue()​方法设置基础值可以影响最终的属性值，从而实现修改玩家攻击力的效果。

Java中的 ​instanceof​操作符，用于检查一个对象是否是由某个类或其子类创建的。
​​“damager instanceof Player”会判断引用变量 ​damager​是否存着 ​Player​类的实例的遥控器。如果是，返回 ​true​；否则，返回 ​false​。
在本例中，​damager​是一个实体对象，可能是玩家或其他类型的实体。通过 ​instanceof​操作符，我们可以判断这个实体是否是玩家，从而进行相应的处理。如果是玩家，就可以获取到对应的 ​Player​对象，并进行一些特定的操作。

getValue()​会返回近战攻击伤害属性的当前值，包括基础值和所有修饰值的总和。 ​​

sendActionBar()​会向指定的玩家发送一个ActionBar消息。在Minecraft中，ActionBar是一种在屏幕经验值上方显示的短文本消息，通常用于向玩家提供一些提示或信息。 ​​

ChatColor.RED​这个常量表示红色的颜色代码，用于给消息设置红色的文本颜色。
