第十三课——数组和字符串也是对象 ​

数组还可以这样创建：

java
int[] arr = new int[5];

数组也是对象，但没有相应的类文件。
5代表数组的长度。

通过这样来为数组赋值元素：

java
arr[0] = 100;
arr[1] = 500;
arr[2] = 300;
arr[3] = 1000;
arr[4] = 700;

相当于：

java
int[] arr = {100, 500, 300, 1000, 700};

以下是第三种创建数组的方式：

java
int[] arr = new int[] {100, 500, 300, 1000, 700};

在Java中，字符串属于引用类型，因此其本质上是一个对象，而不是基本数据类型。
我们可以使用以下两种方式来创建字符串对象：

java
String str1 = "身材不错喔，蛮结实的";
String str2 = new String("都几岁了，还这么害羞");

需要注意的是，字符串在Java中是不可变的，即一旦创建，其值就不能被修改。

可以调用字符串类里的一些方法：

java
String str = "哎呦，你脸红了";
int length = str.length(); // 获取字符串长度
int index = str.indexOf('，'); // 查找逗号的位置
String newStr = str.replace("你", "我"); // 替换字符串

在方法声明中声明一个方法的返回类型，那么就需要在方法的主体中，使用return语句返回值。
任何声明为void的方法都不会返回一个值。它不需要包含return语句，但也可以包含。
return语句从当前方法中退出，并且返回到调用方法的位置。
return语句有两种形式：一种返回值，另一种不返回。
要返回值，只需在 return 关键字后面放置值(或计算值的表达式)即可。如：

java
return ++count;

返回值的数据类型必须与方法声明的返回值的类型匹配。
当方法被声明为 void 时，使用 return 的形式不返回值。如：

java
return;

任何声明为非 void 的方法都必须包含具有相应返回值的 return 语句。
返回值的数据类型必须与方法的声明返回类型相匹配;你不能从声明返回布尔值的方法返回一个整数值。
以上的话用通俗易懂的解释是：

java
public int add() {
 int sum = 1 + 1;
 return sum; // 返回sum的值
}

“sum”是int类型，public int add中间也是int类型，所以可以return sum。
“引用变量名.add()”整体就是sum的值。因为方法返回值，所以“引用变量名.方法()“本身就是返回的值。

void关键字不返回值，方法可以包含return，此时是结束方法执行的用处：

java
public void printNumbers() {
 for (int i = 1; i <= 10; i++) {
 if (i > 6) {
 return; // 如果i大于6，直接结束方法的执行
 }
 System.out.println(i);
 }
}

小结： ​

①学会数组的另外两种创建方法。
②进一步学会使用字符串。
③学会使用return语句。
