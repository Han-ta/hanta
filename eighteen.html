第十八课——向玩家发送屏幕标题 ​

当玩家移动时，在标题栏实时显示玩家的坐标：

java
@EventHandler
 public void onPlayerMove(PlayerMoveEvent event) {
 Player player = event.getPlayer();
 Location location = player.getLocation();
 String message = String.format("x=%.2f, y=%.2f, z=%.2f", location.getX(), location.getY(), location.getZ());
 // 在标题栏上显示消息
 player.sendTitle(message, "", 0, 20, 0);
 }

​​format()​方法是Java中用于格式化字符串的方法，它可以将指定的参数按照指定的格式进行格式化，并返回一个格式化后的字符串：

java
public static String format(String format, Object... args)

其中，​format​ 是一个字符串格式模板，可以包含占位符 ​%s​、​%d​、​%f​ 等等，用于指定参数的插入位置和格式。​args​ 是一个可变参数（即可以传递任意数量的参数，一个都不传也行），表示要插入到格式模板中的实际参数值。​
开头的代码中，"x=%.2f, y=%.2f, z=%.2f"​中的三个“​%.2f​”会分别被替换为X、Y、Z坐标的值。
“​%.2f​”的“f”​表示浮点数，“.2”表示其精度为小数点后两位。
因此，​location.getX()​、​location.getY()​、​location.getZ()​返回的坐标值会被保留两位小数。如果不格式化，将会保留很多位小数。
最终，​String.format()​方法会返回一个包含坐标信息的字符串，比如 ​"x=9.97, y=94.40, z=-41.61"​。

sendTitle()方法用于向玩家发送屏幕标题：

java
void sendTitle(@Nullable
 @Nullable String title,
 @Nullable
 @Nullable String subtitle,
 int fadeIn,
 int stay,
 int fadeOut)

​​各参数表示：
title - 标题文本。
subtitle - 副标题文本。如果不需要显示副标题，可以将该参数设置为空字符串 ​""​。
fadeIn - 标题淡入时间。表示标题从无到有的过渡时间。如果不需要过渡效果，可以将该参数设置为0。以tick为单位。默认值取10。
stay - 标题停留/展示时长。以tick为单位。默认值取70。
fadeOut - 标题淡出时间。表示标题从有到无的过渡时间。如果不需要过渡效果，可以将该参数设置为0。以tick为单位。默认值取20。
