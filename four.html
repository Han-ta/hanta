java
int n = 1 + 2 - 3 * 4 / 5 % 6;

java
int x = 0;
x += 1;

上面两个语句都会将 x 的值递增1。

字符串不是基本数据类型，与char的区别在​​于：char​类型的数据只能存储一个字符，而String​类型的数据可以存储任意长度的字符序列。

一元运算符 ​

+一元加运算符，表示正值(在正数前面，可加可不加)；
-一元减运算符，对表达式求反；
++递增运算符，将值加1；
--递减运算符，将值减1；
!逻辑非运算符，反转布尔值；
java
int m = +1; //结果是1
m = -m; //结果是-1
m++; //结果是0
System.out.println(!true); //结果是false

递增/递减运算符可以在数之前或之后应用：

java
i++;
++i;

上面两个语句结束时i都将加1，但有一点不同：
​​i++;​的作用是先使用变量​i​的值，然后再将变量​i​的值加1。例如，假设i​的初始值为5，

java
​int j = i++;​

执行上面代码后，​j​的值为5，而i​的值变为6。

​​++i;​的作用是先将变量i​的值加1，然后再使用变量​i​的值。例如，假设​i​的初始值为5，

java
​int j = ++i;​

执行上面代码后，​j​的值为6，而​i​的值也为6。

先学习if语句：

java
if (条件) {
 // 如果条件为真，则执行这里的代码块
}

其中，​if​是关键字，后面的括号中是一个布尔表达式，用于判断条件是否成立。如果条件为true，则会执行花括号中的代码块；否则跳过代码块，继续执行后面的代码。
关键字是Java语言中具有特殊含义的单词，它们不能用于变量名、方法名等的命名。

写以下代码，来判断指定坐标的方块是否是空气方块。

java
World world = Bukkit.getWorld("world");
Location loc = new Location(world, x, y, z);
Block block = world.getBlockAt(loc);
// 判断该方块是否为“空气”方块
if (block.getType() == Material.AIR) {
 System.out.println("该位置是空气方块");
} 
if (block.getType() != Material.AIR) {
 System.out.println("该位置不是空气方块");
}

上面是我随便选取一个坐标的结果。

条件运算符 ​

&&​​​表示“与”，当两个操作数都为真（即布尔值均为true）时，整个表达式的值为真；否则为假。
​​||​表示“或”，当两个操作数至少有一个为真时，整个表达式的值为真；否则为假。

例如：

java
boolean a = true;
boolean b = false;
boolean c = true;
if (a && b) {
 // 这里的代码块不会被执行，因为a为真，b为假 
} 
if (a && c) {
 // 这里的代码块会被执行，因为a和c均为真 
}
java
if (a || b) {
 // 这里的代码块会被执行，因为a为真，b为假 
} 
if (b || c) {
 // 这里的代码块会被执行，因为b为假，c为真 
} 
if (b || b) {
 // 这里的代码块不会被执行，因为b为假，两个数均为假 
}

三元运算符的形式如下：
条件 ? 表达式1 : 表达式2
其中，条件是一个布尔类型的表达式，用于判断条件是否成立；
表达式1和表达式2是两个可能的结果，根据条件的真假选择其中一个结果作为整个表达式的值。

java
int a = 10;
int b = 20;
int max = (a > b) ? a : b;
System.out.println("最大值是：" + max);

若a > b成立，则a赋值给max，否则b赋值给max。
