第二十课——创建Boss条 ​

生成boss条，当玩家移动时boss条实时显示玩家的坐标，boss条的血量是玩家的y轴占世界可建筑高度的百分比：

java
public final class MyPlugin extends JavaPlugin implements Listener {
    BossBar bossBar;

    @Override
    public void onEnable() {
        Bukkit.getServer().getPluginManager().registerEvents(this, this);
    }

    @EventHandler
    public void onPlayerJoin(PlayerJoinEvent event) {
        Player player = event.getPlayer();
        // 创建Boss条
        bossBar = Bukkit.createBossBar("", BarColor.GREEN, BarStyle.SEGMENTED_10);
        // 将Boss条添加给玩家
        bossBar.addPlayer(player);
    }

    @EventHandler
    public void onPlayerMove(PlayerMoveEvent event) {
        Location location = event.getPlayer().getLocation();
        // 计算进度条的百分比
        double progress = (location.getY() + 64) / (319.0 + 64);
        if (progress < 0) {
            progress = 0;
        } else if (progress > 1) {
            progress = 1;
        }
        // 更新Boss条的进度和标题
        bossBar.setProgress(progress);
        bossBar.setTitle(String.format("玩家坐标：%.2f, %.2f, %.2f", location.getX(), location.getY(), location.getZ()));
    }
}

​​Bukkit.createBossBar()用于创建Boss条：

java
public static BossBar createBossBar(String title, BarColor color, BarStyle style, BarFlag... flags);

上述各参数的含义是：
​​title​：Boss条的标题，可以使用颜色代码和格式化符号。
​​color​：Boss条的颜色。
​​style​：Boss条分几段。
​​flags​：可变参数。Boss条的标志。
（​​BarFlag.DARKEN_SKY​：这个标志会使得天空变暗。
​​BarFlag.PLAY_BOSS_MUSIC​：这个标志会播放打末影龙时候的BGM。
​​BarFlag.CREATE_FOG​：这个标志会产生雾气。）

addPlayer(player)是让指定玩家的屏幕显示Boss条。

最新MC版本世界可建筑高度为-64到319。计算进度条的百分比时，为了防止出现负数，需要把-64~319整体加64。

​​setProgress(progress)​用于设置Boss条的进度（血量）。进度值必须在0.0到1.0之间，表示Boss条的当前进度百分比。

setTitle()​方法用于设置Boss条的标题。标题可以包含颜色代码和格式化符号。
