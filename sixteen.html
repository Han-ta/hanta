第十六课——生成粒子 ​

当玩家从潜行状态站起来时在腰部位置生成呼啦圈形状的粒子：

java
//玩家切换潜行状态则调用此事件
    @EventHandler
    public void onPlayerToggleSneak(PlayerToggleSneakEvent event) {
        Player player = event.getPlayer();
        World world = player.getWorld();
        // 获得玩家腰部位置（我也不知道腰部离眼睛距离是多少，这里凭感觉写了）
        Location location = player.getEyeLocation().add(0, -0.5, 0);
        //isSneaking()——当玩家从潜行状态站起来时为true
        if (player.isSneaking()) {
            Particle particle = Particle.WAX_OFF;  // 呼啦圈粒子类型
            double radius = 1;  // 呼啦圈半径
            double angle = 0;  // 角度
            // 生成20个呼啦圈粒子
            for (int i = 0; i < 20; i++) {
                double x = location.getX() + radius * Math.cos(angle);
                double y = location.getY();
                double z = location.getZ() + radius * Math.sin(angle);
                world.spawnParticle(particle, x, y, z, 1, 0, 0, 0, 0);
                angle += Math.PI / 10;  // 每次增加18度，即1/20圆周
            }
        }
    }

上面for语句内使用了数学知识参数方程来计算每个呼啦圈粒子的位置，20个呼啦圈粒子在不同位置就构成了呼啦圈。
我不教数学，你们可以自行学习相应数学知识。
​（x ​和 ​z​ 都是关于参数 ​angle​ 的函数，而 ​angle ​则表示当前粒子在呼啦圈上所处的角度。​angle​就是参数，而 ​x​ 和 ​z​ 则是关于 ​angle​ 的函数）

spawnParticle()方法用于生成粒子：

java
void spawnParticle(@NotNull
 @NotNull Particle particle,
 double x,
 double y,
 double z,
 int count,
 double offsetX,
 double offsetY,
 double offsetZ,
 double extra)

以上参数表示：
​​particle​：表示要生成的粒子类型。
​​x​：表示粒子生成的X坐标。
​​y​：表示粒子生成的Y坐标。
​​z​：表示粒子生成的Z坐标。
​​count​：表示要生成的粒子数量。
​​offsetX​：表示粒子在X轴上的偏移量。
​​offsetY​：表示粒子在Y轴上的偏移量。
​​offsetZ​：表示粒子在Z轴上的偏移量。
​​extra​：表示粒子的额外数据，具体含义取决于粒子类型。
