第五课——继续学习if语句 ​

上节课我们写的判断指定坐标的方块是否是空气方块的代码，其实可以改写成：

java
if (block.getType() == Material.AIR) {
 System.out.println("该位置是空气方块");
} else {
 System.out.println("该位置不是空气方块");
}

基本形式是：

java
if (条件) {
 // 如果条件为真，则执行这里的代码块
} else {
 // 如果条件为假，则执行这里的代码块
}

除了上述语句外，还可以使用else if关键字：

java
World world = Bukkit.getWorld("world");
Location loc = new Location(world, x, y, z);
Block block = loc.getBlock();
if (block.getType() == Material.GRASS) {
 System.out.println("该坐标上是草方块");
} else if (block.getType() == Material.DIRT) {
 System.out.println("该坐标上是泥土方块");
} else if (block.getType() == Material.STONE) {
 System.out.println("该坐标上是石头方块");
} else {
 System.out.println("该坐标上是其他方块");
}

它的基本形式如下：

java
if (条件1) {
 // 如果条件1成立，则执行这里的代码块
} else if (条件2) {
 // 如果条件1不成立，且条件2成立，则执行这里的代码块
} else if (条件3) {
 // 如果条件1和条件2都不成立，且条件3成立，则执行这里的代码块
} else if (条件N){
 ......
} else {
 // 如果所有条件都不成立，则执行这里的代码块
}

小结： ​

①学会使用if-else语句。
②学会使用if-else if语句。
