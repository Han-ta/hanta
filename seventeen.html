第十七课——怪物互相攻击 ​

僵尸和骷髅互相打起来：

java
World world = Bukkit.getWorld("world");
Location location = new Location(world, -156, 83, -26);
Zombie zombie = (Zombie) world.spawnEntity(location.add(2, 0, 0), EntityType.ZOMBIE);
Skeleton skeleton = (Skeleton) world.spawnEntity(location.add(-2, 0, 0), EntityType.SKELETON);
// 设置僵尸和骷髅的最大血量和当前血量为500
zombie.setMaxHealth(500);
zombie.setHealth(500);
skeleton.setMaxHealth(500);
skeleton.setHealth(500);
// 设置僵尸的目标为骷髅
zombie.setTarget(skeleton);
// 设置骷髅的目标为僵尸
skeleton.setTarget(zombie);

(Zombie) ​即 (类名) 是Java中的类型转换符，也称为强制类型转换。在这里，​(Zombie)​表示将spawnEntity()​方法返回的实体对象转换为​Zombie​类型。
因为spawnEntity()​方法返回的是Entity​类型的对象，而我们需要将其转换为​Zombie​类型的对象。

小结： ​

①学会调用setTarget()方法。
②知道强制类型转换是什么。
