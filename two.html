第二课——生成一个方块 ​

变量是一种存储数据的容器，可以将它们想象成一个”杯子“，用来存放各种类型的数据。
在方法里的变量叫局部变量。
Java编程语言支持的八种基本数据类型是：

基本数据类型描述
byte整数。它的最小值为-128，最大值为 127(包括)
short整数。它的最小值为-32,768，最大值为 32,767(包括)
int整数。其最小值为-231，最大值为 231-1
long整数。带符号的 long 的最小值为 -263，最大值为 263-1
float浮点数
double浮点数
boolean只有两个可能的值：true 和 false
char代表单个字符

以字母 L 或 l 结尾的数据是long类型的，如233L。
浮点数是一种用于表示实数（包括小数）的数据类型。
以字母 F 或 f 结尾的数据的类型为 float。 double类型的数据可以选择以字母 D 或 d 结尾。
浮点类型(float 和 double)也可以用 E 或 e(科学记数法)，如2.33e666，表示2.33×10666。
char是使用单引号（'）括起来的一种数据，如 'S'、'牛'、'5' 等。单引号里面只有一个字符。

写以下代码，它是用来在指定坐标处生成一个方块：

java
World world = Bukkit.getWorld("world"); //"world"是服务器主世界文件夹的名字。这里我们获得一个世界对象。
Location location = new Location(world, x, y, z); //x, y, z代表指定坐标，请把它换成具体的数字。这里我们获得主世界的一个位置对象。
Block block = world.getBlockAt(location); //获得该位置的方块对象。
block.setType(Material.DIAMOND_BLOCK); //将该位置的方块替换成钻石块。

上图中，请把光标放在红色的代码上，然后点击“导入类”。

如果出现上图中那样的列表，一般应该选择带有“bukkit”字样的那一个选项。上图中，应该选择第一个选项。

java
int x = 8;

意思是声明一个名为“x”的int类型的变量（杯子），里面装着数据“8”。

java
double y = 79.3; //声明一个名为“y”的double类型的变量（杯子），里面装着数据“79.3”。
y = 86.0;

y = 86.0;的意思是把杯子里的“79.3”替换成“86.0”。
