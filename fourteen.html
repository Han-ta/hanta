第十四课——构造函数 ​

一个类除了有实例变量和方法，还有构造函数。
构造函数的语法格式如下：

java
访问修饰符 类名(参数列表) {
 // 构造函数的代码块
}

在Java中，每个类都至少有一个构造函数。如果类里没写构造函数，则Java会提供一个默认构造函数，该构造函数没有参数，且不做任何操作。

构造函数声明看起来像方法声明，但是构造函数使用类的名称并且没有返回类型：

java
public class Person {
 private String name;
 private int age;

 public Person(String name, int age) {
    this.name = name;
    this.age = age;
 }
}

创建对象时，Person后的括号里则按照参数列表填写：

java
Person p = new Person("Tom", 18);

创建对象时会执行构造函数里的代码。
需要注意的是，构造函数可以重载，也就是说，一个类可以定义多个不同参数列表的构造函数。

this.name是实例变量，name是参数：

java
this.name = name;

在Java中，​this​关键字表示​this所在的类的对象的引用。当Person类型的对象p的构造函数执行时，​this表示​this所在的Person类的对象p的引用。
this.name就是对象引用.实例变量。

下面这段代码中，有两个构造函数，一个有参数，一个没有参数：

java
public class Person {
 private String name;
 private int age;
 public Person() {
    this("Tom", 18);  
 }

public Person(String name, int age) {
    this.name = name;  
    this.age = age;  
 }
}

没有参数的构造函数调用另一个构造函数。

构造函数调用另一个构造函数的语句：

java
this("Tom", 18);

它必须放在花括号里的第一行。

在Java中，方法或构造函数的参数列表指的是一组用逗号分隔的参数类型和参数名，用于接收调用者传递给方法或构造函数的数据。
参数列表可以为空，也可以包含一个或多个参数。
​参数类型​是参数的数据类型，​参数名​是参数的名称。

这是一个方法例子：

java
public void print(String a, String b, String c) {
 System.out.println(a + ", " + b + ", " + c);
}

在调用此方法时，我们必须按照相同的顺序传递参数：
引用变量.print("Hello", "world", "Java");
然后"Hello", "world", "Java"分别赋值给变量a，b，c。
// 输出：Hello, world, Java
参数列表在Java中的作用是，它可以接受调用者传递的数据，并在方法体中使用这些数据完成不同的操作。

小结： ​

①学会使用构造函数和this关键字。
②学会使用参数列表。
