第三课——生成一个生物 ​

现在我们来学习数组。
数组就像是一组杯子，可以用来存储相同类型的多个数据。每个数据都有一个编号，也就是索引，从0开始逐渐递增。

java
int[] num = {1, 2, 3, 4, 5};

在int后添加“[]”代表创建一个int类型的数组变量，变量名为num。
共有5个杯子，分别装有”1“、”2“、”3“、”4“、”5“。
数组的长度（杯子的数量）是在创建数组时确立的。创建后，其长度是固定的。比如上面的数组长度为5。

接下来我们在指定坐标生成一个巨人僵尸。

java
World world = Bukkit.getWorld("world");
Location location = new Location(world, x, y, z);
world.spawnEntity(location, EntityType.GIANT); //在主世界指定位置生成一个巨人僵尸。

我们创建了一个double类型的数组，装着坐标。
xyz[0]、xyz[1]、 xyz[2]都是变量名+括号的形式。
xyz[0]意思是获得数组中索引为0的元素，也就是第一个元素。
xyz[1]是获得第二个元素。
xyz[2]是获得第三个元素。

结果：

java
int[] num = {1, 2, 3, 4, 5};
num[2] = 98;

num[2] = 98;的意思是将原本的“3”替换成“98”。
最终num数组是{1, 2, 98, 4, 5}。

小结： ​

①知道数组是什么。
②学会生成一个生物。
